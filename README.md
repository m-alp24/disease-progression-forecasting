# Disease Progression Forecasting

## 📌 Overview
This project analyses and forecasts disease incidence using time series and machine learning models to support public health decision-making.

## 📊 Methods
- ARIMA (time series model)
- XGBoost (machine learning model)

## 🛠 Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels

## 📈 Results
XGBoost achieved better predictive performance than ARIMA, particularly in capturing complex patterns in disease trends.
